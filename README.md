# 📊 Sales Performance Dashboard – Power BI

## 📌 Project Overview

The **Sales Performance Dashboard** is an interactive **Power BI** project designed to analyze sales performance, customer behavior, profitability, and business trends. The dashboard enables business stakeholders to monitor key performance indicators (KPIs), identify high-performing products and regions, understand customer purchasing patterns, and make data-driven business decisions through interactive visualizations.

---

## 🎯 Project Objective

The objective of this project is to build an interactive **Power BI Dashboard** that provides a comprehensive analysis of sales, profit, customers, products, and business performance using a retail sales dataset.

---

## 📂 Dataset Information

- **Dataset:** Retail Sales Dataset
- **Time Period:** January 2026 – June 2026
- **Tool Used:** Microsoft Power BI

---

## 🛠️ Technologies Used

- Microsoft Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)
- Data Modeling
- Interactive Dashboard Design
- KPI Cards
- Charts & Visualizations
- Slicers & Filters

---

# 📈 Dashboard Pages

The dashboard consists of four interactive pages:

### 1️⃣ Executive Summary
Provides an overview of overall business performance through key KPIs.

### 2️⃣ Sales Analysis
Analyzes sales trends across products, categories, brands, cities, and time.

### 3️⃣ Customer Analysis
Examines customer demographics, purchasing behavior, payment preferences, and ratings.

### 4️⃣ Profit Analysis
Focuses on profitability, profit margins, costs, and profit contribution across different business segments.

# 💡 Project Insights

- Electronics generated the highest overall sales and profit.
- Laptops were the best-selling product.
- Hyderabad recorded the highest sales.
- Customers aged **41–50 years** contributed the highest number of purchases.
- Sales showed a downward trend during the selected period.
- The overall business achieved a **16.19% Profit Margin**.
